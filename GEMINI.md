# GEMINI.md - 포트폴리오 프로젝트 (안건)

이 프로젝트는 UX/UI 디자이너 **안건(An Gun)**의 개인 포트폴리오 웹사이트입니다. 고품질의 시각적 디자인, 부드러운 애니메이션, 그리고 현대적인 사용자 경험(UX)에 중점을 둔 정적 웹 애플리케이션입니다.

## 프로젝트 개요

*   **목표:** 디자인 프로젝트, 기술 역량 및 경력을 시각적으로 매력 있게 소개합니다.
*   **타겟 대상:** 디자인 및 IT 업계의 잠재적 클라이언트와 채용 담당자.
*   **주요 기술:**
    *   **HTML5/CSS3:** 시맨틱 구조 및 CSS 변수 활용.
    *   **GSAP & ScrollTrigger:** 고도의 인터랙티브 애니메이션 및 스크롤 효과.
    *   **Lenis:** 부드러운 스크롤(Smooth Scroll) 엔진.
    *   **SplitType:** 텍스트 분할 애니메이션 유틸리티.
    index,html을 제외한 다른 파일은 각 확장자에 맞게 폴더로 분류해.

## 작업지침 (Working Guidelines) - 중요

이 프로젝트를 수정하거나 기능을 추가할 때 AI 에이전트는 다음 지침을 엄격히 준수해야 합니다.

1.  **애니메이션 표준:**
    *   모든 인터랙션은 **GSAP**을 사용하여 `script.js`에서 관리합니다.
    *   스크롤 애니메이션은 `ScrollTrigger`를 사용하며, 성능 최적화를 위해 `scrub` 또는 `stagger` 옵션을 적절히 활용합니다.
2.  **배경색 및 그라데이션 (Color Morphing):**
    *   피그마 디자인을 따른다.
    *   1
3.  **반응형 및 모바일 최적화:**
    *   `clamp()` 함수를 사용하여 폰트 크기와 간격을 유연하게 관리합니다.
    *   터치 기기(`isTouchDevice`) 여부를 감지하여 데스크탑 전용 효과(커스텀 커서, 자기장 효과 등)가 모바일에서 오작동하지 않도록 처리합니다.
    *   768px 이하에서는 모바일 전용 햄버거 메뉴와 오버레이 네비게이션이 활성화되어야 합니다.
4.  **코드 청결성:**
    *   리플릿(Replit) 등 특정 플랫폼에 종속된 주석이나 코드를 배제하고, 순수 정적 웹 표준을 따릅니다.
    *   CSS 변수를 `:root`에서 관리하여 시각적 일관성을 유지합니다.

## 프로젝트 구조

*   `index.html`: 메인 구조 및 섹션별 `data-bg`, `data-theme` 속성 포함.
*   `style.css`: 테마별 스타일, 그라데이션 변수, 반응형 레이아웃 정의.
*   `script.js`: Lenis 스크롤, GSAP 애니메이션, 커서 효과, 배경 모핑 로직.
- **프론트엔드**: React 19, Vite, Tailwind CSS v4, React Router v7, FontAwesome
- **백엔드**: Python
## 빌드 및 실행

*   **로컬 실행:** `index.html`을 브라우저에서 직접 실행하거나 로컬 서버(`static-web-server` 등)를 사용합니다.
*   **배포:** 정적 호스팅 서비스(GitHub Pages, Vercel 등)를 이용합니다.

# 안건 Portfolio

이 프로젝트는 신입 UIUX 디자이너 *안건*의 개인 포트폴리오 사이트야.
나는 코딩 배운지 6개월 됐어. 하지만 대부분의 코드는 이해하고 짤 수 있는 상황이야.
코드를 짤때 공통 코딩 컨벤션을 기초로 해서 만들어.
구직용 포트폴리오야.

## 공통 코딩 컨벤션

- **들여쓰기**: 2 spaces (탭 금지)
- **문자열**: 작은따옴표 (`'`)
- **세미콜론**: JS/JSX에서 생략
- **파일 네이밍**: 컴포넌트·페이지는 PascalCase (`.jsx`), 훅·유틸은 camelCase (`.js`)
- **TypeScript 사용 금지** — `.jsx`와 `.js`만 사용
- **라우터**: React Router v7 Data Mode만 허용 (`createBrowserRouter` + `RouterProvider`). `BrowserRouter`/`Routes`/`Route` 선언적 모드 금지
- **Tailwind**: v4 유틸리티 클래스만 사용, 임의값 `[...]` 지양 — `@theme` 토큰 사용
- **커밋 메시지 접두사**: `feat:`, `fix:`, `docs:`
